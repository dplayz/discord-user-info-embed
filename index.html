dp's discord-user-profile-embed-v2

A project that embeds Discord profile data, powered mainly by the Lanyard API and VueJS.

What powers this?

I'm grateful to these underlying projects and frameworks for making this project possible:

APIs

  • Lanyard's WebSocket API (ws://api.lanyard.dev)
  • dcdn.dstn.co

UI

  • VueJS (vuejs.org)
  • Iframe Resizer (https://github.com/davidjbradshaw/iframe-resizer)
  • marked (https://github.com/markedjs/marked)

and for the numerous sources I used to make it possible
